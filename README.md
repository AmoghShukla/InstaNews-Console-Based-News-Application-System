# Console-Based News Application System

A menu-driven Python console application that fetches and manages news articles using the NewsAPI (Everything Endpoint) based on user-provided keywords.

---

## Features

- Search and fetch news articles using keywords  
- Filter out incomplete articles (missing title or author)  
- Store valid articles locally in JSON format  
- Generate a summary file containing article titles and descriptions  
- View stored article titles  
- Delete specific articles by title  
- Exit the application gracefully  

---

## Tech Stack

- Python  
- Requests  
- JSON  
- python-dotenv (for API key management)
